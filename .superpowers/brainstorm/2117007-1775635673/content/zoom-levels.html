Map Data Visualization — Zoom Levels

How should emission data appear as the user zooms in and out?

A

Cluster → Marker → Plume Outline

3-level progressive disclosure:

  • Zoom 1–5 (global) — clustered circles with count badge, colored by dominant gas type
  • Zoom 6–11 (regional) — individual dot markers, sized by emission rate
  • Zoom 12+ (local) — actual plume polygon outline rendered from TIFF geometry
