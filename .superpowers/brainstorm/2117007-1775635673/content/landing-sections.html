Landing Page Sections

Which sections should the public landing page include? Select all that apply.

1

Hero — Full-width banner

Large headline, subtitle, animated globe or map background, "Open Map" CTA button. First thing visitors see.

2

Key Stats Bar

Animated counters: total detections, countries covered, gas types monitored, latest detection date. Quick credibility signal.

3

Gas Types Overview

Cards for CH4, CO, NO2 — brief description of each gas, why it matters, detection method. Educational section.

4

Latest Detections Feed

A scrollable list or grid of the most recent plume detections — satellite, location, gas type, flux rate. Live data feel.

5

Satellite / Data Sources

Logos and brief descriptions of the satellites used (e.g. GAOFEN-5-01A). Builds trust and scientific credibility.

6

About / Mission Statement

Short paragraph about the project's purpose, who runs it, and why it matters. Footer-adjacent or standalone section.

Select all sections you want. Hero is recommended as a must-have.
